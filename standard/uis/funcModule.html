引用方式

  • 引用方式一、 

    								// ====安装&引入====
								npm install cy_static
								import funcModule from "cy_static/funcModule";
								// ====使用====
								funcModule("#test", {
						                    "keyId": "quickEntry",
						                    "backgroundColor": "#ffffff",
						                    onBefore: function(data, param) {
						                        if (data.length > 5) {
						                            param.row = Math.ceil(data.length / 2);
						                        } 
						                    }
						                });

    引用方式二、 

    								<script type="text/javascript" src="/static/js/libs/require2.3.6.js"></script>
								<script type="text/javascript">
									requirejs(["funcModule"], function(funcModule) {
									    funcModule("#test", {
								                    "keyId": "quickEntry",
								                    "colSpace": 15,
								                    "row": 5,

								                    "paddingTop": 10,

								                    "paddingLeft": 10,

								                        if (data.length > 5) {
								                            param.row = Math.ceil(data.length / 2);
								                        } 
								                    }
								                });																    
									});
								</script>

方法参数

  • params 参数
    参数名 类型 默认值 描述 必需
    keyId String 广告推荐位 "" Y
    rowSpace String 列表行高度 5 N
    row String 每行的列数 5 N
    column String 功能球行数 2 N
    paddingTop String 模块上边距 10 N
    paddingBottom String 模块下边距 10 N
    paddingLeft String 模块左边距 10 N
    paddingRight String 模块右边距 10 N
    unLazyNum String 不懒加载图片 0 N
    backgroundColor String 模块背景色 #ffffff N
    moduleSpm String 模块SPM编码 "" N
    moduleName String 模块名称 "" N
    data Object||Array 本地数据[] "" N
    onBefore Function 渲染前处理 function(data, params, dom) {} N
    onAfter Function 渲染后处理 function(data, params, dom) {} N
