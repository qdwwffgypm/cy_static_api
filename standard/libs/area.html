构造方法

  • area(params : Object) : Object

    支持级联滑动（如地区）与非级联滑动(日期选择)等；

    参数名 类型 默认值 描述 必需
    elem String||Object "" 绑定点击显示元素 N
    isArea Boolean true 是否级联查询 N
    data Array [] 非常级联时的数据 N
    emptyNum Number 2 选择item前面空几个选项 N
    maxNum Number 100 最大显示级联数量 N
    text String text 数量文本属性 N
    value String value 数量值属性 N
    items Array [] 各级联项属性 N
    ajaxFun Function null 级联时查询数据调用的方法 N
    onShow Function null 插件显示时的回调 N
    onCancel Function null 点击取消时的回调 N
    onSubmit Function null 点击确认时的回调 N 
    										// items选项说明 
										{//  
											id : "", // 当前项对应的input选择器
											value : "",// 当前项对应的input默认选中值
											css: "",// 当前项样式类
											readonly: false// 当前项是否只读
										}
											
    										// ajaxFun选项说明 
										{//  
											params:{
												index: 0,// 当前第几项
							        			value: 0// 当前选中项的值
											},// 
											callback: Function,// 设置回调用数据
										}

    把滚动条滑动到固定位置；

  • onSubmit(func : Function) : Void

    把滚动条滑动到固定位置；
