查看源码： /static/js/libs/page.js

PS：该对象在CY对象下；

引用方式

  • 引用方式一、 

    								// ====安装&引入====
								npm install cy_static
								import page from "cy_static/page";
								// ====使用====
								 page("#test", {
								        count: 5,
								        page: 1,
								        total: 80
								    })

    引用方式二、 

    								<script type="text/javascript" src="/static/js/libs/require2.3.6.js"></script>
								<script type="text/javascript">
									requirejs(["page"], function(page) {
									    page("#test", {
									        count: 5,
									        page: 1,
									        total: 80
									    })
									})
								</script>

    引用方式三、 

    									<script type="text/javascript" src="/static/js/libs/offset.js"></script>
									<script type="text/javascript" src="/static/js/libs/tooltip.js"></script>
									<script type="text/javascript" src="/static/js/libs/select.js"></script>
									<script type="text/javascript" src="/static/js/libs/page.js"></script>
									<script type="text/javascript">
									    CY.page("#test", {
									        count: 5,
									        page: 1,
									        total: 80
									    })
									</script>

构造方法

  • page(dom : Object|selector,params : Object) : Object

    设置page的值，如果dom为page标签，自动加载里面的数据

    params 参数
    参数名 类型 默认值 描述 必需
    count Number 5 显示页数 N
    page Number 1 当前页面 N
    total Number 1 总记录条数 N
    pageSize Number 10 分页条数 N
    showSep Boolean false 显示分隔线 N
    hasPageInfo Boolean false 显示分页详情 N
    hasPage Boolean false 显示页数 N
    hasPageInput Boolean false 显示输入框 N
    pageInfoAlign String "right" 显示分页详情位置left|right N
    pageInputAlign String "center" 显示输入框位置left|center|right N
    sizeInputAlign String "center" 显示输入分页位置left|center|right N
    sizeInputArr Array [5, 10] 显示输入分页 N
    url String "" 跳转URL N
    target String "_self" URL打开方式 N
    disabled String "disabled" 没有分页样式 N
    onSwitch Function null 切换事件 N
    onPageChange Function function () {} 点击页数时触发 N
    onSizePageChange Function function () {} 选择页时触发 N
