查看源码： /static/js/libs/offset.js

PS：该方法在CY对象下；

获取元素位置信息

引用方式

  • 引用方式一、 

    								// ====安装&引入====
								npm install cy_static
								import offset from "cy_static/offset";
								// ====使用====
								var pos = offset("#test");
								// 获得元素离顶部距离
								pos.top
								// 获得元素离左边距离
								pos.left

    引用方式二、 

    								<script type="text/javascript" src="/static/js/libs/require2.3.6.js"></script>
								<script type="text/javascript">
									requirejs(["offset"], function() {
									       var pos = offset("#test");
										// 获得元素离顶部距离
										pos.top
										// 获得元素离左边距离
										pos.left																				    
									});
								</script>

    引用方式三、 

    									<script type="text/javascript" src="/static/js/libs/offset.js"></script>
									<script type="text/javascript">
									       var pos = CY.offset("#test");
										// 获得元素离顶部距离
										pos.top
										// 获得元素离左边距离
										pos.left																					    
									</script>

构造方法

  • offset(dom : dom|String) : Object

    添加迭代器加载模块

    返回值对象含义
    key 说明
    top 远素离窗口top距离
    left 远素离窗口left距离
