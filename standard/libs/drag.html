查看源码： /static/js/libs/drag.js

PS：该对象在CY对象下；

拖曳功能，将左边元素拖到右边元素内

引用方式

  • 引用方式一、 

    								// ====安装&引入====
								npm install cy_static
								import drag from "cy_static/drag";
								// ====使用====
								    CY.drag({
								        menuLayout: "#left", // 左边布局元素
								        menuItem: "item", // 模块元素样式名称
								        pageLayout: "#right", // 右边容器
								    })

    引用方式二、 

    								<script type="text/javascript" src="/static/js/libs/require2.3.6.js"></script>
								<script type="text/javascript">
									require(["/static/js/libs/drag.js"], function(drag) {
									    CY.drag({
									        menuLayout: "#left", // 左边布局元素
									        menuItem: "item", // 模块元素样式名称
									        pageLayout: "#right", // 右边容器
									    })
									})
								</script>

    引用方式三、 

    									<script type="text/javascript" src="/static/js/libs/offset.js"></script>
									<script type="text/javascript" src="/static/js/libs/drag.js"></script>
									<script type="text/javascript">
									    CY.drag({
									        menuLayout: "#left", // 左边布局元素
									        menuItem: "item", // 模块元素样式名称
									        pageLayout: "#right", // 右边容器
									    })
									</script>

构造方法

  • drag(params : Object) : Object

    将左边元素拖到右边元素内

    params 参数
    参数名 类型 默认值 描述 必需
    menuLayout dom|String "" 左边布局元素 Y
    menuItem String "" 模块元素样式名称 Y
    pageLayout dom|String "" 右边容器 Y

实例方法

  • addStartHooks(func : Function) : Void

    拖动开始触发的事件

  • addMoveHooks(func : Function) : Void

    拖动中触发的事件

  • addStopHooks(func : Function) : Void

    拖动结束触发的事件

  • addClickPageItem(func : Function) : Void

    已拖到右边的元素的点击事件
