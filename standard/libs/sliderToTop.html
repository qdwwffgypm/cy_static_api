构造方法

  • sliderToTop(params : Object) : Object

    把正常元素滑动到固定位置的顶部；

    参数名 类型 默认值 描述 必需
    elem String||Object "" 窗口元素 Y
    resetElem String||Object "" 复位元素--存在的话只有他能复位 N
    time Number 400 滑动时间(毫秒) N
    top Number 0 离顶部的距离 N
    resetWidth Number 30 重置复位高度 N
    onStart Function function (height, event, params) {} 拖动开始 N
    onScroll Function function (height) {} 滚动事件 N
    onMove Function function (height, event, params) {} 拖动中 N
    onStop Function function (height, event, params) {} 拖动结束 N

实例方法

  • pause() : Void

    禁止滑动；

  • play() : Void

    启动滑动；

  • onArriveTop(func: Function) : Void

    滑动到顶回调；

  • onArriveBot(func: Function) : Void

    滑动到底回调；

  • destroy() : Void

    销毁实例里所有事件；
