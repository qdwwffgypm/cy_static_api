vue项目规范

查看源码： /static/js/libs/tooltip.js

PS：该对象在CY.tooltip 对象下；
1、给指定元素绑定提示层；

引用方式

  • 引用方式一、 

    								// ====安装&引入====
								npm install cy_static
								import tooltip from "cy_static/tooltip";
								// ====使用====
								CY.tooltip("#test", {
								    content: "如需帮助请打10086",
								    eventType: "click"
								})

    引用方式二、 

    								<script type="text/javascript" src="/static/js/libs/require2.3.6.js"></script>
								<script type="text/javascript">
									requirejs([ "/static/js/libs/tooltip.js" ], function(tooltip) {
										tooltip("#test", {
										    content: "如需帮助请打10086",
										    eventType: "click"
										})
									});
								</script>

    引用方式三、 

    									<script type="text/javascript" src="/static/js/libs/tooltip.js"></script>
									<script type="text/javascript">
										CY.tooltip.tooltip("#test", {
										    content: "如需帮助请打10086",
										    eventType: "click"
										});	
									</script>

构造方法

  • tooltip(dom : selector|String, params : Object) : Object

    给元素绑定提示层；

    示例
    参数名 类型 描述 必需
    dom Object|String 要绑定提示层的元素 Y
    params Object 提示层相关参数 Y
    params 参数
    参数名 类型 默认值 描述 必需
    content dom|String "" 提示层内容 Y
    delay Number "0" 延迟时间 N
    width Number 100 提示层宽 N
    offset Array [0, 0] 位置的偏移量 B
    disabled Boolean false 是否可用 N
    eventType String "hover" 事件类型hover|click N
    placement String "bottom" 出现位置 :
    top
    right
    bottom
    left
    top left
    top center
    top right
    right top
    right center
    right bottom
    bottom left
    bottom center
    bottom right
    left top
    left center
    left bottom
    		N
    className String "" 样式名 N
    timeout Number 500 离开隐藏间隔时间 N
    callback Function function(win) {} N
    onShow Function function(win) {} 显示时触发的函数 N
    onHide Function function(win) {} 隐藏时触发的函数 N
    onClose Function function(win) {} 销毁时触发的函数 N
    onClick Function function(win) {} 点击时触发的函数 N

实例方法

  • show() : String

    显示图层；

  • hide() : String

    隐藏图层；

  • close() : String

    销毁图层；

  • setDisabled(disabled : Boolean) : String

    开户与禁用提示层；
