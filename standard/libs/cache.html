查看源码： /static/js/libs/cache.js

PS：该对象在CY对象下；
1、cache对象主要是对数据的缓存处理；
2、缓存方式localStorage；不支持或无痕模式下采用对象形式；
3、默认缓存时间30分钟；
4、进入页面时执行一次清除过期数据；

引用方式

  • 引用方式一、 

    								// ====安装&引入====
								npm install cy_static
								import cache from "cy_static/cache";
								// ====使用====
								cache.set("key", "value", 3600);

    引用方式二、 

    								<script type="text/javascript" src="/static/js/libs/require2.3.6.js"></script>
								<script type="text/javascript">
										requirejs(["cache"], function(cache) {
										    cache.set("key", "value", 3600);
										});
								</script>

    引用方式三、 

    									<script type="text/javascript" src="/static/js/libs/cache.js"></script>
									<script type="text/javascript">
											CY.cache.set("key", "value", 3600);
									</script>

属性

  • debug : Boolean

    是否开启测试模式；测试模式不缓存数据；默认false；

  • localStorage : Boolean

    检测当前环境是否支持localStorage存储；

数据存储方法

  • set (key:String, val:Any, time:Number=1800000) : Void

    设置要要缓存的数据；time毫秒数

  • get (key:String) : Any

    获取缓存数据；

  • runClear() : Void

    手动执行清除已过期的数据；

  • del (key:String) : Void

    删除缓存数据；

图像存储方法

  • toCacheImg (img:Image, time:Number=1800000) : Boolean

    缓存图片对象；

  • getBase64Img (img:Image) : String

    将图片对象对为base64位字符串；
