1、根据秒数生成倒计时HTML模板字符串；

引用方式

  • 引用方式一、 

    								// ====安装&引入====
								npm install cy_static
								import countdown from "cy_static/countdown";
								// ====使用====
								countdown.run({
									time : 60*60,
									downText : "距离结束还有{1}时{2}分{3}秒",
									onChange : function(html) {
										console.log(html);
									},
									callback : function() {
										console.log("倒计时结束");
									},
								});

实例方法

  • format(str : String, arr : Array) : String

    输出模版字符串；

    示例
    参数名 类型 默认值 描述 必需
    str String "字符串模板" Y
    arr Array "序列化数组" Y 
    										// 示例1
										requirejs([ "/static/js/libs/countdown.js" ], function(countdown) {
											var text = countdown.format("用户名：{0}, 邮箱：{1}",['张三','123456@qq.com']);
											console.log(text); // 用户名：张三, 邮箱：123456@qq.com
										});

  • run(params : Object) : Void

    输出倒计时HTML模版字符串；

    示例
    params 参数
    参数名 类型 默认值 描述 必需
    time Number "0" 倒计时间秒 Y
    downText String "" 倒计文本字符串模板 Y
    onChange Function function() {} 改变触发事件 N
    callback Function function() {} 倒计完成事件 N 
    											// 示例1
											requirejs([ "/static/js/libs/countdown.js" ], function(countdown) {
												countdown.run({
													time : 60,
													downText : "{3}秒后重新发送",
													onChange : function(html) {
														console.log(html);
													},
													callback : function() {
														console.log("重新发送");
													},
												});
											});
											
    											// 示例2
											requirejs([ "/static/js/libs/countdown.js" ], function(countdown) {
												countdown.run({
													time : 60*60,
													downText : "距离结束还有{1}时{2}分{3}秒",
													onChange : function(html) {
														console.log(html);
													},
													callback : function() {
														console.log("倒计时结束");
													},
												});
											});
