vue项目规范

查看源码： /static/js/libs/sliderTab.js

PS：该对象在CY.sliderTab 对象下；
1、tab标签菜单，支持滑动切换、点击回调，动态改变当前选中值并居中显示；
2、视例结构:
<div id="myTab">
  <div class="content">
    <div class="item">热门</div>
    <div class="item">美食餐饮</div>
  </div>
</div>

引用方式

  • 引用方式一、 

    								// ====安装&引入====
								npm install cy_static
								import sliderTab from "cy_static/sliderTab";
								// ====使用====
								// 用法一
								sliderTab({
									elem: "#myTab",		    
									currIndex: 0,// 窗口元素--要设置高度
									content: ".content",// 滚动元素
									onClick: function(index) {
										console.log("当前===" + index);
									},// 拖动开始
									onStart: function(left) {
										console.log("拖动开始===" + left);
									},// 拖动中
									onMove: function(left) {
										console.log("拖动中===" + left);
									},// 滚动中
									onScroll: function(left) {
										console.log("'%c滚动中-----" + left,'color:#f00;')
									}, // 拖动结束
									onStop: function(left) {
										console.log("拖动结束===" + left);
									}
								});

构造方法

  • sliderTab(params : Object) : Object

    tab标签菜单，支持滑动切换、点击回调，动态改变当前选中值并居中显示；

    参数名 类型 默认值 描述 必需
    elem String||Object "" 窗口元素 Y
    content String||Object "" 窗口子元素 Y
    time Number 1000 滑动时间(毫秒) N
    drag Number 0.1 阻尔糸数 N
    moveInt Boolean true 移动整格 N
    isSpring Boolean true 启动弹性吸附效果 N
    resetWidth Number 10 重置复位宽度(米) N
    currCss String "curr" 默认选中样式 N
    currIndex Number 0 默认选中的窗口 N
    isOverMove Boolean true 超过窗口是否能滑动 N
    onStart Function function (left, event, params) {} 拖动开始 N
    onScroll Function function (left) {} 滚动事件 N
    onMove Function function (left, event, params) {} 拖动中 N
    onStop Function function (left, event, params) {} 拖动结束 N
    onClick Function function (index) {} tab点击事件 N

实例方法

  • pause() : Void

    禁止滑动；

  • play() : Void

    启动滑动；

  • reset() : Void

    重置操作/让已选中的Tab显示在可视窗口内；

  • setCurrIndex(index : Number) : Void

    设置要选中Tab下标（从0开始）；并在可视窗口中显示
