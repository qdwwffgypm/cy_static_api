查看源码： /static/js/libs/sliderWin.js

PS：该对象在CY.sliderWin 对象下；
1、窗口滑动切换、暂停、动态设置当前窗口；
2、视例结构:
<div id="myWin">
  <<div class="item">窗口1</div>
  <<div class="item">窗口2</div>
</div>

  • 引用方式一、

									}

构造方法

  • sliderWin(params : Object) : Object

    窗口滑动切换、暂停、动态设置当前窗口；

    参数名 类型 默认值 描述 必需
    elem String||Object "" 窗口元素 Y
    content String||Object "" 窗口子元素 Y
    time Number 1000 滑动时间(毫秒) N
    springDrag Number 0.8 超出窗体阻尔糸数 N
    resetWidth Number 40 重置复位宽度(px) N
    currIndex Number 0 默认选中的窗口 N
    isOverMove Boolean true 超过窗口是否能滑动 N
    onStart Function function (left, event, params) {} 拖动开始 N
    onScroll Function function (index, dom, left) {} 滚动事件 N
    onMove Function function (left, event, params) {} 拖动中 N
    onStop Function function (left, event, params) {} 拖动结束 N
    onChange Function function (index) {} 窗口改变事件 N

实例方法

  • pause() : Void

    禁止滑动；

  • play() : Void

    启动滑动；

  • getCurrIndex() : Number

    获得当前激活的窗口

  • setActivated(isActivated) : Void

    是否激活；

  • destroy() : Void

    销毁实例里所有事件；

  • setCurrIndex(index : Number) : Void

    设置要选中窗口下标（从0开始）；并在可视窗口中显示
