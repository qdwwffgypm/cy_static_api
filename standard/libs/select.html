引用方式

  • 引用方式一、 

    								// ====安装&引入====
								npm install cy_static
								import select from "cy_static/select";
								// ====使用====
								 select("#test", {
								     data: [{
								         value: "001",
								         text: "中国"
								     }, {
								         value: "002",
								         text: "日本"
								     }]
								 })

    引用方式二、 

    								<script type="text/javascript" src="/static/js/libs/require2.3.6.js"></script>
								<script type="text/javascript">
										requirejs(["select"], function(select) {
										    select("#test", {
										        data: [{
										            value: "001",
										            text: "中国"
										        }, {
										            value: "002",
										            text: "日本"
										        }]
										    })
										})
								</script>

    引用方式三、 

    									<script type="text/javascript" src="/static/js/libs/offset.js"></script>

构造方法

  • select(dom : Object|selector,params : Object) : Object

    设置select的值，如果dom为select标签，自动加载里面的数据

    params 参数
    参数名 类型 默认值 描述 必需
    width Number 100 宽度 N
    attrText String "text" data数据里的文本字段 N
    attrValue String "value" data数据里的value字段 N
    selected String "" 选中项的value N
    data Array [] 要显示的数据 N

实例方法

  • setValue(value : String) : Void

    设置选中项

  • setData(data : Array) : Void

    设置数据

  • setDisabled(disabled : Boolean) : Void

    开户或禁用

  • onChange(func : Function) : Void

    选中事件
