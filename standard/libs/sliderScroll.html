构造方法

  • sliderScroll(params : Object) : Object

    上下滑动窗口，暂停(锁定), 恢复正常，滑动到固定位置等；

    参数名 类型 默认值 描述 必需
    elem String||Object "" 窗口元素 Y
    content String||Object "" 窗口子元素 Y
    time Number 1000 滑动时间(毫秒) N
    drag Number 0.6 阻尔糸数 N
    springDrag Number 0.85 超出窗体阻尔糸数 N
    scrollTop Number 0 默认滚动条位置 N
    isSpring Boolean true 启动弹性吸附效果 N
    isTopEffect Boolean false 顶部动画效果 N
    isBotEffect Boolean true 底部动画效果 N
    hasScrollbar Boolean true 显示滚动条 N
    isLockfull Boolean false 锁定全屏状态(取消系统默认滚动) N
    pullDown Object null 下拉刷新(顶部刷新加载)对象{} N
    pushDown Object null 上拉刷新(底部刷新加载)对象{} N
    onOverflow Function function (height) {} 滑动超出窗口高度 N
    onTopOrBottom Function function (height, event, params) {} 已到顶部OR底部 N
    onStart Function function (height, event, params) {} 拖动开始 N
    onScroll Function function (height) {} 滚动事件 N
    onMove Function function (height, event, params) {} 拖动中 N
    onStop Function function (height, event, params) {} 拖动结束 N

事件

  • scroll() : Void

    触发绑定在content滚动事件；

实例方法

  • pause() : Void

    禁止滑动；

  • play() : Void

    启动滑动；

  • scrollTop(index : Number) : Void

    把滚动条滑动到固定位置；

  • setLockfull(isLock : Boolean) : Void

    设置全屏锁定状态；

  • setLockTopSpace(isLock: Boolean) : Void

    锁定必须滚动top到顶再释放默认事件；

  • setLockBotSpace(isLockf : Boolean) : Void

    锁定必须滚动bottom到底再释放默认事件；

  • destroy() : Void

    销毁实例里所有事件；

  • isEndPull(isEnd : Boolean) : Void

    下拉滚动加载异步数据时，用于设置滚动内容加载状态(加载完还是还有下一页)超时15秒自动设置为还有下一页

  • isEndPush(isEnd : Boolean) : Void

    上拉滚动加载异步数据时，用于设置滚动内容加载状态(加载完还是还有下一页)超时15秒自动设置为还有下一页
