查看源码： /static/js/libs/lazyloadUtil.js

PS：该对象在CY对象下；

浮动固定与滚动选中

引用方式

  • 引用方式一、 

    								// ====安装&引入====
								npm install cy_static
								import lazyloadUtil from "cy_static/lazyloadUtil";
								// ====使用====
								lazyloadUtil();

    引用方式二、 

    								<script type="text/javascript" src="/static/js/libs/require2.3.6.js"></script>
								<script type="text/javascript">
									requirejs(["lazyloadUtil"], function(lazyloadUtil) {
									    lazyloadUtil();																	    
									});
								</script>

    引用方式三、 

    									<script type="text/javascript" src="/static/js/libs/offset.js"></script>
									<script type="text/javascript" src="/static/js/utils/lazyloadUtil.js"></script>
									<script type="text/javascript">
										CY.lazyloadUtil();																		    
									</script>

构造方法

  • lazyloadUtil(params : Object) : Void

    滚动加载图片

    params 参数
    参数名 类型 默认值 描述 必需
    step Number 100 间距加载图片 N
    realSrc String "realSrc" 真实路径 N
    container dom|String "body" 滚动元素 N
