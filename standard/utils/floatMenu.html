查看源码： /static/js/utils/floatMenu.js

PS：该对象在CY对象下；

当页面滑动到固定位置时，把元素设为浮动并固定在特定位置

引用方式

  • 引用方式一、 

    								// ====安装&引入====
								npm install cy_static
								import floatMenu from "cy_static/floatMenu";
								// ====使用====
								floatMenu("#test", {top:50});

    引用方式二、 

    								<script type="text/javascript" src="/static/js/libs/require2.3.6.js"></script>
								<script type="text/javascript">
									requirejs(["floatMenu"], function() {
									    CY.floatMenu.float("#test", {top:50});																	    
									});
								</script>

    引用方式三、 

    									<script type="text/javascript" src="/static/js/libs/offset.js"></script>
									<script type="text/javascript" src="/static/js/utils/floatMenu.js"></script>
									<script type="text/javascript">
										CY.floatMenu.float("#test", {top:50});																		    
									</script>

构造方法

  • floatMenu(dom : selector|String, params : Object) : Void

    浮动元素

    params 参数
    参数名 类型 默认值 描述 必需
    top Number 0 离开滚动元素上边框距离 N
    container dom|String window 滚动元素 N
    onShowFloat Function function() {} 当元素从普通元素变为浮动时触发 N
    onHideFloat Function function() {} 当元素从浮动元素变为普通浮动时触发 N
    onChange Function function() {} 滚动时触发的函数 N
