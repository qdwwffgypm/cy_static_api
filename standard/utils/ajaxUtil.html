vue项目规范

查看源码： /static/js/utils/ajaxUtil.js

PS：该对象在CY对象下；
1、该方法底层使用XMLHttpRequest、ActiveXObject实现；
2、仅支持："json", "text", "script", "xml" 类型； "jsonp", "arraybuffer", "blob", "document", "stream"需另外封装；
3、ajaxUtil.js内部依赖CryptoJS.js，browser.js，和ajax.js；

引用方式

  • 引用方式一、 

    								// ====安装&引入====
								npm install cy_static
								import ajaxUtil from "cy_static/ajaxUtil";
								// ====使用====
								ajaxUtil.loadData("xxxx/xxx.do", {
									method:"POST",
									headers:{
										"Content-Type":"application/json"
									}
								}).then(function(data) {
									// code
								})

    引用方式二、 

    								<script type="text/javascript" src="/static/js/libs/require2.3.6.js"></script>
								<script type="text/javascript">
										requirejs(["ajaxUtil"], function(ajaxUtil) {
										    ajaxUtil.loadData("xxxx/xxx.do", {
												method:"POST",
												headers:{
													"Content-Type":"application/json"
												}
											}).then(function(data) {
												// code
											})
										});
								</script>

    引用方式三、 

    									<script type="text/javascript" src="/static/js/libs/CryptoJS.js"></script>
									<script type="text/javascript" src="/static/js/libs/browser.js"></script>
									<script type="text/javascript" src="/static/js/libs/ajax.js"></script>
									<script type="text/javascript" src="/static/js/utils/ajaxUtil.js"></script>
									<script type="text/javascript">
											CY.ajaxUtil.loadData("xxxx/xxx.do", {
												method:"POST",
												headers:{
													"Content-Type":"application/json"
												}
											}).then(function(data) {
												// code
											})
									</script>

方法

  • ajaxUtil : Object

    ajax帮助类；

  • ajaxUtil.loadData(url : String, request : Object = {}) : Promise

    方法通过 HTTP 请求加载远程数据；请求数据经过MD5加密：用法与ajax相同

    示例
    request 参数
    参数名 类型 描述
    Content-Type String 默认："application/json"
    standard Boolean 是否使用标准格式传送数据；
    默认：false
