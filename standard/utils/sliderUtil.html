查看源码： /static/js/utils/sliderUtil.js

PS：该对象在CY对象下；

浮动固定与滚动选中

引用方式

  • 引用方式一、 

    								// ====安装&引入====
								npm install cy_static
								import sliderUtil from "cy_static/sliderUtil";
								// ====使用====
								sliderUtil({
						                  elem:"#adModule040752366199529444",
						                  content:".content",
						                  isLoop:false,
						                  autoTime : 5000
						                });

    引用方式二、 

    								<script type="text/javascript" src="/static/js/libs/require2.3.6.js"></script>
								<script type="text/javascript">
									requirejs(["sliderUtil"], function(sliderUtil) {
									    sliderUtil({
								                  elem:"#adModule040752366199529444",
								                  content:".content",
								                  isLoop:false,
								                  autoTime : 5000
								                });																	    
									});
								</script>

    引用方式三、 

    									<script type="text/javascript" src="/static/js/libs/offset.js"></script>
									<script type="text/javascript" src="/static/js/utils/sliderUtil.js"></script>
									<script type="text/javascript">
										CY.sliderUtil({
								                  elem:"#adModule040752366199529444",
								                  content:".content",
								                  isLoop:false,
								                  autoTime : 5000
								                });																	    
									</script>

构造方法

  • sliderUtil(params : Object) : Void

    滚动加载图片

    params 参数
    参数名 类型 默认值 描述 必需
    delay Number 800 延迟轮播时间 N
    drag Number 0.1 阻尔糸数 N
    time Number 1000 滑动时间 N
    resetWidth Number 10 重置复位宽度 N
    autoTime Number 0 自动播放时间（毫秒） N
    attr String 'left' 滚动元素属性[left|top] N
    realSrc String "realSrc" （如果有）延迟加载图片的真实路径 N
    moveInt Boolean true 移动整数 N
    isOverMove Boolean true 超过窗口是否能滑动 N
    isSpring Boolean true 启动弹性吸附效果 N
    isLoop Boolean false 是否循环轮播 N
    isShowNum Boolean false 显示x/xx比例数量如:1/10 N
    elem dom|String "" 滑动元素选择器 N
    content dom|String "" 滚动元素选择器 N
    navMenu (dom|String)|Boolean "" 底部菜单小点;如果存在选择器或dom直接读取，否则自动生成 N
    onChange Function function () {} 滚动改变事件 N
