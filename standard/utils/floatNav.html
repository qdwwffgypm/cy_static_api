浮动固定与滚动选中

引用方式

  • 引用方式一、 

    								// ====安装&引入====
								npm install cy_static
								import floatNav from "cy_static/floatNav";
								// ====使用====
								floatNav.float("#test", {top:50});

    引用方式二、 

    								<script type="text/javascript" src="/static/js/libs/require2.3.6.js"></script>
								<script type="text/javascript">
									requirejs(["floatNav"], function() {
									    CY.floatNav.float("#test", {top:50});																	    
									});
								</script>

    引用方式三、 

    									<script type="text/javascript" src="/static/js/libs/offset.js"></script>
									<script type="text/javascript" src="/static/js/utils/floatNav.js"></script>
									<script type="text/javascript">
										CY.floatNav.float("#test", {top:50});																		    
									</script>

anchor构造方法

  • anchor(params : Object) : Void

    滚动选中相关元素

    params 参数
    参数名 类型 默认值 描述 必需
    menuItem dom|String '' 固定元素选择器多个','分隔 N
    elemItem dom|String '' 内容元素选择器多个','分隔 N
    goTop dom|String '' 返回顶部元素选择器 N
    currCss String 'curr' 选中固定元素样式 N
    speeds Array [0] 内容元素修正偏移量多个','分隔 N
    moveTime Number 300 点击时内容元素移动时间 N
    deferTime Number 1000 延迟时间毫秒数 N
    onSelect Function function(num, isClick) {} 选中回调函数 N 
    										// 示例1
										floatNav.anchor({
							                            menuItem: "#goodsTitle .l-item",
							                            elemItem: "#goodsRecList > div"
							                        });

float构造方法

  • float(dom : selector|String, params : Object) : Void

    浮动元素

    params 参数
    参数名 类型 默认值 描述 必需
    top Number 0 离开滚动元素上边框距离 N
    container dom|String window 滚动元素 N
    onChange Function function() {} 滚动时触发的函数 N
